题目 A | 项目编号 1923 | 【自编教材】实验2-1判断是奇数还是偶数

题目描述

 给定一个整数，编写一个程序判断它是奇数还是偶数。

输入

从键盘输入一个整数n (-1000 <= n <= 1000)。

输出

如果输入的整数n是偶数，输出”偶数”；如果是奇数，输出”奇数”。

样例输入

7

样例输出

奇数

题解

题目 B | 项目编号 1924 | 【自编教材】实验2-2判断是否为闰年

题目描述

给定一个年份，编写一个程序判断它是否为闰年。

输入

从键盘输入一个整数年份n（0 <= n <= 10000）。

输出

如果输入的年份n是闰年，输出”yes”；如果不是闰年，输出”no”。

样例输入

2024

样例输出

yes

提示

闰年的判断条件：如果年份可以被4整除但不能被100整除，或者可以被400整除，就是闰年。

题目 C | 项目编号 1925 | 【自编教材】实验2-3计算上网费

题目描述

某上网服务提供商的上网费用计算方法如下：

同时，为了鼓励多上网，每月收费最多不超过180元。如果计算得到的费用超过180元，将按照180元计算。
编写一个程序，根据上网时间计算上网费用，并考虑费用不超过180元的情况。

输入

从键盘输入一个整数，表示上网时间（小时）（1 <= 上网时间 <= 100）。

输出

输出一个小数（保留2位小数），表示上网费用，考虑不超过180元的限制。

样例输入

30

样例输出

80.00

题目 D | 项目编号 1930 | 【自编教材】实验2-4判断三角形并计算面积

题目描述

编写一个程序，从键盘输入三个正数，代表三角形的三边长 a, b, c。程序需要判断是否可以构成一个三角形。如果可以构成三角形，计算并输出三角形的面积；如果不能构成三角形，输出”不能构成三角形”。

输入

输入三个正数 a, b, c（1 ≤ a, b, c ≤ 1000）

输出

如果可以构成三角形，输出三角形的面积（保留两位小数），否则输出”不能构成三角形”

样例输入

3 4 5

样例输出

能构成，面积为: 6.00

提示

三角形能够构成的条件为：任意两边之和大于第三边。使用海伦公式计算三角形的面积。
如果面积大于0，则可以构成三角形

题目 E | 项目编号 1931 | 【自编教材】实验2-5三个数排序

题目描述

编写一个程序，从键盘输入三个整数，然后显示以下菜单：

  1. 从大到小排序
  2. 从小到大排序

用户选择菜单选项后，程序将对输入的三个数进行相应排序，并输出排序结果。

输入

三个整数 a, b, c (-1000 ≤ a, b, c ≤ 1000)
一个整数表示菜单选项（1或2）

输出

如果用户选择菜单选项1，输出排序后的三个数从大到小，用空格分隔。
如果用户选择菜单选项2，输出排序后的三个数从小到大，用空格分隔。

样例输入

5 3 8
1

样例输出

8 5 3

题目 G | 项目编号 1933 | 实验2-7输出被3和5整除的数

题目描述

编写程序，将1~200之间整数中能同时被3和5整除的数输出出来，要求每行输出10个。

输入

无需输入。

题目 H | 项目编号 1934 | 【自编教材】实验2-8判断几位数

题目描述

编写一个程序，输入一个正整数，然后求它是几位数。

输入

一个正整数N（1 <= N <= 10^9）。

输出

一个整数，表示输入的正整数N的位数。

样例输入

12345

样例输出

5

提示

可以通过循环不断将输入的正整数除以10，每除一次，位数加1，直到被除数变为0为止。循环的次数即为位数。

题目 I | 项目编号 1935 | 【自编教材】实验2-9数列求和

题目描述

编写一个程序，计算数列的总和：11 + 111 + 211 + 311 + … + 2011。数列中的每一项都是从前一项递增100而来。

输入

无需输入。

输出

输出数列的总和。

样例输出

21231

提示

可以使用循环来计算数列的总和。在循环中，从11开始，每次加上100，直到达到2011为止，然后将每一项累加起来。

题目 K | 项目编号 1937 | 【自编教材】实验2-11根据线索求出车号

题目描述

一辆卡车违反交通规则，撞人后逃跑。现场有三人目击事件，但都没有记住车号，只记下车号的一些特征：

  • 甲说：牌照的前两位数字是相同的。
  • 乙说：牌照的后两位数字是相同的，但与前两位不同。
  • 丙是数学家，他说：四位的车号刚好是一个整数的平方。
    编写一个程序，根据以上线索求出车号。

输入

无需输入。

输出

输出一个4位整数，表示车号。

样例输出

7744

提示

可以使用循环遍历4位整数，分别检查是否满足甲、乙和丙的条件。如果满足所有条件，即为答案。

题解

题目 L | 项目编号 1938 | 【自编教材】实验2-12多少种兑换方案

题目描述

有100元人民币，需要兑换成10元、5元和1元的纸币，每一种纸币至少一张，共20张纸币。编写一个程序，计算共有多少种兑换方案，以及每种方案兑换多少张纸币。

输入

一个正整数N（1 <= N <= 10^9）。

输出

输出两行：
前几行每行输出一个兑换方案的详细信息，格式为 “10元 x张 5元 x张 1元 x张”，其中 x 分别表示该方案中10元、5元和1元纸币的数量，每种纸币的数量至少为1。
最后一行输出一个整数，表示共有多少种兑换方案。

提示

10元 4张 5元 11张 1元 5张
10元 8张 5元 2张 1元 10张
2

题解

题目 M | 项目编号 1939 | 【自编教材】实验2-13求满足条件的最大n

题目描述

编写一个C语言程序，求满足以下条件的最大的正整数n：
2^1 + 2^2 + 2^3 + … + 2^n 小于1000。
请编写一个程序，找到满足上述条件的最大的n，并输出它。

输入

无需输入。

输出

一个整数，表示满足条件的最大的n。

样例输出

9

提示

你可以使用循环来逐步增加n的值，计算2^1 + 2^2 + 2^3 + … + 2^n的累加和，直到累加和小于1000为止。然后，输出找到的最大的n。
注意：2^n 表示2的n次方，即2的n次方等于2乘以2乘以2…，共乘以n个2。

题解

题目 N | 项目编号 1940 | 【自编教材】实验2-14求最大公约数和最小公倍数。

题目描述

从键盘输入两个正整数，求其最大公约数和最小公倍数。

输入

从键盘输入两个正整数，每个整数都不超过1000。

输出

程序应输出最大公约数和最小公倍数的值。

样例输入

12
18

样例输出

6
36

提示

你可以使用辗转相除法来计算最大公约数（GCD），然后使用公式 (a * b) / GCD 计算最小公倍数（LCM）。

题目 O | 项目编号 1941 | 【自编教材】实验2-15判断素数

题目描述

从键盘输入一个正整数，判断其是不是素数。

输入

从键盘输入一个正整数，不超过10000。

输出

程序应输出判断结果，要么是，要么不是。

样例输入

17

样例输出

是

提示

素数是只有两个正因子（1和自身）的正整数。你可以使用循环来检查是否有除了1和自身以外的因子，如果有，则不是素数。

题目 P | 项目编号 1942 | 【自编教材】实验2-16输出三角字符阵列

题目描述

下面程序的功能是输入任意整数给n后，输出n行由大写字母A开始构成的三角形字符阵列图形。例如，输入整数5时(注意：n不得大于10)，程序运行结果如下：
A B C D E
F G H I
J K L
M N
O

输入

从键盘输入一个整数n，其中1 <= n <= 10。

输出

根据输入的整数n，输出由大写字母A开始构成的三角形字符阵列图形。

样例输入

5

样例输出

A B C D E
F G H I
J K L
M N
O

提示

可以使用嵌套循环来生成三角形字符阵列图形。外循环用于控制行数，内循环用于控制每行的字母输出。可以使用ASCII码来生成字母序列。
